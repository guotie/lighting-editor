# lighting editor

an out of box richtext editor base on slate

# demo

https://guotie.github.io/lighting-editor/

# usage

install:

```
npm i lighting-editor
```


```
import { LightingEditor } from 'lighting-editor'

function MyEditor() {
    return (
    <div>
        <LightingEditor content={initialValue} />
    </div>
  );
}

```

see example.
